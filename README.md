# Solda&Cia — Assistência Técnica

Site institucional fictício desenvolvido para a disciplina de Front End.

## Objetivo

Criar um site para uma assistência técnica especializada em celulares e computadores utilizando HTML, CSS e JavaScript.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## Seções do Site

* Início
* Sobre
* Serviços
* Contato

## Funcionalidades

* Navegação entre seções
* Exibição dos serviços oferecidos
* Formulário de contato com validação dos campos
* Mensagem de sucesso exibida após o envio, simulando o salvamento dos dados em um banco de dados
* Layout responsivo para diferentes dispositivos
